# Happy-Android
使用玩Android api。实现一个RxJava + Retrofit + MVVM的项目。通俗易懂
* okhttp+retrofit+rxjava+mvvm
* rxlifecycle解决RxJava内存泄漏
* banner
* glide
* svg
* 沉浸式状态栏
* cookieJar
* eventbus
* dataBinding
* aop实现登录校验（只要是重复代码，建议使用aop解决）
